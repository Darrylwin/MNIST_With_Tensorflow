# Démo MNIST - Deep Learning

Reconnaissance de chiffres manuscrits avec TensorFlow.

## Installation



```bash
#créer l'environnement virtuel et l'activer
python -m venv .venv
.\.venv\Scripts\activate

#installer les dependances
pip install -r requirements.txt
```

## Exécution

```bash
python app.py
```

Appuyez sur ENTRÉE pour avancer étape par étape.
